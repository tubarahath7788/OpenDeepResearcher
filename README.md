# OpenDeepResearcher

## Milestone 1 - Foundation Setup

### Objective
Set up environment, architecture, and stubbed research flow.

### Features Completed
- Python virtual environment setup
- Modular project structure
- Planner, Searcher, Writer agents (stubbed)
- LM Studio integration
- Tavily API integration
- Basic research pipeline working

### How to Run

1. Install dependencies:
   pip install -r requirements.txt

2. Add .env file with Tavily key

3. Start LM Studio local server

4. Run:
   python main.py
